Identifiants utilisés pour scanner le projet séléctionné et ses repositories Git.

Note : Seuleument les identifiants "username + password" sont supportés. Les autres types d'identifiants seront filtrés. La raison est que Jenkins va interroger le serveur OrangeForge via les API, et que seul basic auth est implémenté.

Si le projet contient des repositories privés, alors les identifiants spécifiés doivent permettre l'accès à ces repositories.
