Identifiants utilisés pour scanner les repositories Git et ses branches et check out-er les sources.

Note : Seuleument les identifiants de type "username + password" sont supportés. Les autres types d'identifiants seront filtrés. La raison est que Jenkins va interroger le serveur Tuleap via les API, et que seul basic auth est implémenté.

Si le projet contient des repositories privés, alors les identifiants spécifiés doivent permettre l'accès à ces repositories.
